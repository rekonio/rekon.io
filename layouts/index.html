Mapping

From a single domain name we provide:

  • associated subdomains, aliases, etc.
  • IP addresses & active ports
  • providers: CDN, Cloud, etc.
  • services: HTTP(S), FTP, SSH, etc.
  • load balancing and reverse proxying logic
  • etc.

Analysis

From the gathered resources we run a full analysis:

  • proper HTTP services setup: redirection, headers, etc.
  • TLS: certificates, expiration, etc.
  • SSH: weak configuration, obsolete version, etc.
  • unknown running services
  • subdomains takeovers, typosquatting
  • etc.
